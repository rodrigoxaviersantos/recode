# O nome desse processo é CRUD, um acrônimo para:

````
*Create* Inserir registros. 

*Read* Ler ou consultar os registros.

*Update* Atualizar os dados da coluna. 

*Delete* Remover um registro.

Os recursos e comandos do SQL são disponibilizados e agrupados em cinco subgrupos:

> DML (Data Manipulation Language)

"Linguagem de manipulação de dados" que nos permite utilizar os principais comandos do CRUD: SELECT, INSERT, UPDATE, DELETE (selecionar, inserir, atualizar e remover, respectivamente). Grande parte dos comandos que usamos diariamente são desse grupo.

> DQL (Data Query Language)

"Linguagem de Consulta de Dados", é um subconjunto do DML que contém apenas o comando SELECT. Esta subdivisão ocorreu pois o SELECT não manipula os dados, apenas consulta, por mais que o SELECT ainda continue no grupo do DML.

> DDL (Data Definition Language)

"Linguagem de Definição de Dados" que nos permite definir estruturas do banco, tais como criar as tabelas, seus tipos de dados e propriedades usando apenas comandos, como fizemos de forma visual usando as IDEs.

> DCL (Data Control Language)

"Linguagem de Controle de Dados" que nos permite usar comandos para conceder e revogar acessos aos objetos. Este grupo de comandos será abordado em momento oportuno.

> DTL (Data Transaction Language)

"Linguagem de Transação dos Dados", são os comandos que nos permitem criar transações das operações. Este grupo de comandos também será abordado em momento oportuno.
